# Development

## Setup
cp _env .env
cp _development_env .development_env
##

## Database setup

```bash
cd rock
scripts/container_shell.sh schema
rake db:drop db:create db:migrate db:setup
```

## Build

```bash
cd rock
cd api-service
./gradlew clean build -x test
```

## Run

```bash
cd rock
scripts/development_up.sh ui

http://localhost:8111/ui
```
